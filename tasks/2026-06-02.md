# Atividade Prática sobre escrita e leitura binária em arquivos

## Exercício 1 — Cadastro inicial

**Enunciado.**  
Crie um programa em C que cadastre 5 clientes em um arquivo binário chamado `clientes.dat`. Cada cliente deve ter: número da conta, nome e saldo. O programa deve ler os dados do teclado e gravar cada registro com `fwrite()`.

## Exercício 2 — Leitura completa

**Enunciado.**  
A partir do arquivo `clientes.dat` criado no exercício anterior, faça um programa que leia todos os registros e mostre na tela em formato de tabela. Use `fread()` para recuperar os dados.

## Exercício 3 — Releitura com rewind

**Enunciado.**  
Crie um programa que abra `clientes.dat`, leia e mostre o primeiro registro, depois use `rewind()` para voltar ao início do arquivo e mostre novamente o primeiro registro. O objetivo é verificar o reposicionamento do ponteiro de arquivo.

## Exercício 4 — Atualização de registro

**Enunciado.**  
Amplie o programa para permitir atualizar o saldo de um cliente específico. O usuário deve informar o número da conta, e o programa deve usar `fseek()` para ir diretamente até o registro correspondente, ler o dado com `fread()`, alterar o saldo e gravar o registro de volta com `fwrite()`.

## Exercício 5 — Sistema completo

**Enunciado.**  
Crie um sistema de manutenção de contas com menu contendo estas opções:

1. Cadastrar um novo cliente em uma posição específica.
2. Consultar um cliente pelo número da conta.
3. Atualizar o saldo de um cliente.
4. Listar todos os clientes.
5. Restaurar a leitura do arquivo do início com `rewind()` para repetir a listagem.
6. Encerrar.

O arquivo deve ser binário, com registros de tamanho fixo. Para acessar posições específicas, use `fseek()`. Para ler e escrever registros, use `fread()` e `fwrite()`. Para repetir a leitura do início, use `rewind()`.
